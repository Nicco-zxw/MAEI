MAEI
This repo hosts the code for paper "Mining high average-efficiency itemsets based on a compact list structure", Gufeng Li, xuanwei Zhang, Tao Shang Shuo Chen, 2025

Requirements
Java programming language.

Running the program
A simple way is to run MainTestHAEI

Benchmarks
The data format used is the same as in the file DB_Utility.txt and Stock.txt.
Additional datasets like DBUtility.txt can be accessed from SPMF;
Additional datasets like Stock.txt can be generated by GenInvest.java.



Introduction
The MAEI algorithms are proposed to efficiently mine high average-efficiency itemsets.
